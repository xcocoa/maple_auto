## 使用说明

### 运行应用
```bash
python main.py config/roguelike.yaml
```

### 配置优化参数
编辑 `config/roguelike.yaml`:
- 调整 `game.max_combat_duration` 改变战斗超时
- 调整 `game.npc_find_timeout` 改变NPC搜索超时
- 调整 `navigation.stuck_threshold` 改变卡住灵敏度

### 查看日志
日志输出到 `logs/` 目录，按时间戳命名：
```
logs/20250324_120000.log
```

### 性能监控
启用性能监控需设置日志级别为 DEBUG（查看详细指标）
