# MapleAuto 用户手册

## 一、应用简介

### 什么是 MapleAuto

MapleAuto 是一款基于 Python 脚本驱动的 Android 自动化框架。它允许用户编写 Python 脚本来控制手机执行自动化操作，包括模拟手势点击、滑动、截图找图、UI 节点查找与操控、文本输入等。应用内置代码编辑器、悬浮窗控制面板和完善的日志系统，适用于重复性任务自动化、UI 测试等场景。

### 主要功能概述

- **脚本管理**：导入、编辑、运行和删除 Python 自动化脚本
- **悬浮窗控制**：通过悬浮窗实时控制脚本的播放、暂停、停止
- **内置编辑器**：支持 Python 语法高亮的代码编辑器，可直接保存并运行
- **截图与找图**：屏幕截图、模板匹配找图、找色、取色
- **UI 节点操作**：通过无障碍服务查找和操控界面元素
- **手势模拟**：点击、长按、滑动、多点滑动、缩放手势
- **脚本设置面板**：JSON 驱动的动态配置界面
- **插件系统**：支持通过 APK 加载扩展插件

### 系统要求

- Android 8.0 (API 26) 及以上
- 需开启悬浮窗权限、无障碍服务、屏幕录制权限

## 二、安装与初始设置

### 安装 APK

将 MapleAuto 的 APK 文件传输到手机，点击安装即可。如系统提示"未知来源"，需在设置中允许安装。

### 首次启动权限配置

首次启动时，应用会显示权限引导页面（Setup Guide），通过分步向导引导你完成以下配置：

**1. 悬浮窗权限（Overlay Permission）**
允许 MapleAuto 在其他应用上层显示悬浮控制面板。点击"Enable"跳转至系统设置页面开启。

**2. 无障碍服务（Accessibility Service）**
启用 MapleAuto 的无障碍服务，用于模拟点击、滑动以及读取屏幕 UI 节点信息。在系统"无障碍"设置中找到 MapleAuto 并开启。

**3. 屏幕录制权限（Screen Capture）**
授权屏幕截图能力，脚本可使用截图进行图像识别和自动化操作。

**4. 输入法设置（可选）**
配置 MapleAuto 输入法用于高级文本输入自动化。此步骤为可选项。

引导完成后点击"Done"进入主界面。

## 三、主界面说明

### 底部导航栏

主界面采用底部导航栏设计，包含三个页签：

| 页签 | 图标 | 功能 |
|------|------|------|
| **Scripts** | 代码图标 | 脚本列表，管理和运行脚本 |
| **Logs** | 列表图标 | 查看脚本运行日志 |
| **Settings** | 齿轮图标 | 权限管理与引擎设置 |

顶部标题栏显示 "MapleAuto" 以及引擎连接状态指示器（绿色圆点 = 已连接，红色圆点 = 未连接）。

### 脚本列表页

显示已导入的所有脚本，每个脚本卡片展示名称、描述和版本号。点击播放按钮可直接运行脚本。长按脚本卡片弹出菜单，可选择运行、编辑或删除。列表为空时显示提示"Tap + to import a Python script"。右下角的浮动按钮"Run Script"用于运行当前选中的脚本。

### 日志页

实时显示脚本运行日志，支持按级别过滤（All / Info / Warning / Error）。每条日志前有颜色圆点标识级别（绿色=Info、黄色=Warning、红色=Error），并附带精确到毫秒的时间戳。支持自动滚动到最新日志，也可手动浏览。点击右上角清除按钮可清空所有日志。

### 设置页

分为三个区域：
- **Permissions**：查看并管理各项权限状态，未授权的权限可点击"Enable"开启
- **Engine Settings**：引擎配置项，包括开机自启动引擎、保持活跃、默认脚本超时时间
- **About**：应用版本信息

## 四、脚本管理

### 导入脚本

在脚本列表页点击右下角"+"按钮，选择 `.py` 文件或 `.zip` 脚本包导入。

### 查看脚本详情

点击脚本卡片可查看脚本详情并将其设为当前脚本。

### 编辑脚本

长按脚本卡片，在弹出菜单中选择"Edit"，进入内置编辑器。

### 删除脚本

长按脚本卡片，在弹出菜单中选择"Delete"即可删除脚本。

### 脚本包格式说明

脚本包为 `.zip` 格式，推荐目录结构如下：

```
my_script/
  manifest.json   — 脚本元信息（名称、版本、描述等）
  main.py         — 主入口脚本
  config.json     — 脚本设置面板配置（可选）
  assets/         — 资源文件目录（模板图片等）
```

## 五、悬浮窗操作

### 显示/隐藏

运行脚本时悬浮窗自动显示，关闭时自动隐藏。悬浮窗在所有应用上层显示，方便随时控制脚本。

### 按钮功能

| 按钮 | 功能 |
|------|------|
| 播放/暂停 | 运行中时显示暂停图标，暂停或空闲时显示播放图标 |
| 停止 | 立即停止脚本执行 |
| 设置 | 打开脚本设置面板（如脚本提供了 config.json） |
| 关闭 | 关闭悬浮窗 |

悬浮窗通过状态指示条显示当前状态：灰色=空闲、蓝色=加载中、绿色=运行中、黄色=已暂停、红色=出错。

### 拖动与自动收起

悬浮窗支持自由拖动。静止 5 秒后自动收起为小圆点（迷你模式），点击可重新展开。迷你模式下圆点颜色对应当前脚本状态。

### 拖到底部删除

拖动悬浮窗至屏幕底部的删除区域，松手后关闭悬浮窗。

## 七、脚本设置面板

### JSON 配置文件格式

脚本可通过 `config.json` 定义设置面板，面板从屏幕底部弹出，用户修改参数后保存，脚本内通过 `get_config()` 读取。

```json
{
  "title": "脚本设置",
  "tabs": [
    {
      "name": "基础设置",
      "items": [
        {"type": "switch", "key": "auto_retry", "label": "自动重试", "default": true},
        {"type": "edittext", "key": "retry_count", "label": "重试次数", "default": "3", "inputType": "number"},
        {"type": "slider", "key": "delay", "label": "操作延迟(ms)", "min": 100, "max": 3000, "default": 500}
      ]
    },
    {
      "name": "高级",
      "items": [
        {"type": "spinner", "key": "mode", "label": "运行模式", "options": ["标准", "快速", "精确"], "default": "标准"},
        {"type": "checkbox", "key": "debug", "label": "调试模式", "default": false}
      ]
    }
  ]
}
```

### 支持的控件类型

| 类型 | 说明 |
|------|------|
| `checkbox` | 复选框，布尔值 |
| `switch` | 开关，布尔值 |
| `edittext` | 文本输入框，支持 `inputType`（text/number/numberDecimal/phone/textPassword） |
| `spinner` | 下拉选择器，通过 `options` 定义选项列表 |
| `slider` | 滑块，需设置 `min`、`max`、`default` |
| `label` | 纯文本标签，仅用于分组说明 |

## 八、Python 脚本 API 参考

脚本中通过 `from autopilot import *` 导入所有 API。

### 8.1 手势操作

```python
click(x, y, duration_ms=100, random_range=0)
```
点击指定坐标。`random_range > 0` 时添加随机偏移。返回 `bool`。

```python
long_press(x, y, duration_ms=1000, random_range=0)
```
长按指定坐标。返回 `bool`。

```python
swipe(x1, y1, x2, y2, duration_ms=500)
```
从 `(x1,y1)` 滑动到 `(x2,y2)`。返回 `bool`。

```python
swipe_up(duration_ms=500)
swipe_down(duration_ms=500)
swipe_left(duration_ms=500)
swipe_right(duration_ms=500)
```
快捷方向滑动，从屏幕 75% 位置滑至 25% 位置。返回 `bool`。

```python
multi_swipe(points, duration_ms=500)
```
沿多个路径点 `[(x1,y1), (x2,y2), ...]` 依次滑动。返回 `bool`。

```python
pinch_in(center_x, center_y, start_distance=200, end_distance=50, duration_ms=500)
pinch_out(center_x, center_y, start_distance=50, end_distance=200, duration_ms=500)
```
缩放手势。`pinch_in` 为缩小，`pinch_out` 为放大。返回 `bool`。

### 8.2 屏幕截图与图像识别

```python
screenshot()
```
截取当前屏幕。返回 `bytes` 图像数据，失败返回 `None`。

```python
screenshot_size()
```
返回截图尺寸 `(width, height)`。

```python
find_image(template_path, threshold=0.9)
```
在屏幕中查找模板图片。`template_path` 为模板图片绝对路径，`threshold` 为匹配阈值 (0.0-1.0)。返回 `{"x": int, "y": int, "confidence": float}` 或 `None`。

```python
find_color(color, left=0, top=0, right=0, bottom=0, threshold=10)
```
在指定区域查找匹配颜色的像素。`color` 为整数颜色值（如 `0xFFFF0000`），`threshold` 为色差容忍度 (0-255)。返回 `(x, y)` 或 `None`。

```python
get_pixel(x, y)
```
获取指定坐标的像素颜色。返回 `int`（0xAARRGGBB 格式）。

### 8.3 UI 节点操作

```python
find_node(text=None, id=None, cls=None, desc=None)
```
查找第一个匹配的 UI 节点。返回包含 `text`、`id`、`className`、`description`、`clickable`、`enabled`、`left`、`top`、`right`、`bottom` 属性的 `dict`，未找到返回 `None`。

```python
find_all_nodes(text=None, id=None, cls=None, desc=None)
```
查找所有匹配的 UI 节点。返回 `list[dict]`。

```python
click_node(text=None, id=None, cls=None, desc=None)
```
查找并点击匹配的节点。返回 `bool`。

```python
long_click_node(text=None, id=None, cls=None, desc=None)
```
查找并长按匹配的节点。返回 `bool`。

```python
set_node_text(new_text, text=None, id=None, cls=None, desc=None)
```
设置节点的文本内容。返回 `bool`。

```python
wait_for_node(text=None, id=None, cls=None, desc=None, timeout_s=10.0, interval_s=0.5)
```
等待匹配的节点出现，超时返回 `None`。

```python
scroll_forward() / scroll_backward()
```
向前/向后滚动（默认执行上滑/下滑）。

### 8.4 系统操作

```python
press_home()      # 返回桌面
press_back()      # 返回上一页
press_recents()   # 打开最近任务
```
模拟系统按键。返回 `bool`。

```python
open_app(package_name)
```
通过包名启动应用，如 `open_app("com.example.app")`。

```python
shell(command)
```
执行 Shell 命令，返回标准输出字符串。

```python
input_text(text)
```
向当前焦点输入框输入文本。返回 `bool`。

### 8.5 文本输入

```python
input_text(text)     # 通过无障碍 API 输入文本

press_tab()          # 按 Tab 键
```

### 8.6 工具函数

```python

```
等待指定秒数（支持小数），暂停期间自动阻塞。

```python
log(message)
```
记录日志到应用日志页面。

```python

```
显示 Android Toast 提示。

```python

```
读写脚本配置（对接设置面板）。

```python

```
为数值添加 +/- offset_range 范围内的随机偏移。

## 九、脚本示例

### 示例1：简单自动点击脚本

```python
from autopilot import *

log("开始自动点击")
for i in range(10):
    click(540, 960)
    log(f"第 {i+1} 次点击完成")
    sleep(1)
log("任务结束")
```

### 示例2：找图点击脚本

```python
from autopilot import *

target = find_image("/data/local/tmp/assets/button.png", threshold=0.85)
if target:
    log(f"找到目标: ({target['x']}, {target['y']}), 置信度: {target['confidence']:.2f}")
    click(target["x"], target["y"])
else:
    log("未找到目标图片")
```

### 示例3：UI 节点操作脚本

```python
from autopilot import *

open_app("com.example.app")
sleep(3)

login_btn = wait_for_node(text="登录", timeout_s=10)
if login_btn:
    click_node(text="登录")
    sleep(1)
    set_node_text("user@example.com", id="com.example.app:id/input_email")
    press_tab()
    input_text("password123")
    click_node(text="确认")
    log("登录操作完成")
else:
    log("未找到登录按钮")
```

### 示例4：带配置面板的脚本

`config.json`：
```json
{
  "title": "自动签到设置",
  "items": [
    {"type": "edittext", "key": "repeat", "label": "重复次数", "default": "5", "inputType": "number"},
    {"type": "slider", "key": "interval", "label": "间隔(秒)", "min": 1, "max": 30, "default": 3},
    {"type": "switch", "key": "notify", "label": "完成后通知", "default": true}
  ]
}
```

`main.py`：
```python
from autopilot import *

repeat = int(get_config("repeat", 5))
interval = int(get_config("interval", 3))
notify = get_config("notify", True)

log(f"开始执行，重复 {repeat} 次，间隔 {interval} 秒")
for i in range(repeat):
    click(540, 960)
    log(f"第 {i+1}/{repeat} 次操作")
    sleep(interval)

if notify:
    toast("自动签到完成！")
log("全部完成")
```

## 十、常见问题 (FAQ)

### 无障碍服务被系统关闭怎么办

部分手机系统（如 MIUI、EMUI）在省电优化时会自动关闭无障碍服务。解决方案：
- 在系统设置中将 MapleAuto 加入电池优化白名单
- 开启应用的"自启动"权限
- 在设置页开启"Keep alive"和"Auto-start engine on boot"选项

### 悬浮窗无法显示

确认已在系统设置中授予 MapleAuto "显示在其他应用上层"权限。部分系统需要在应用管理中单独开启。

### 截图功能不可用

- 确认已授予屏幕录制权限（每次重启应用后可能需要重新授权）
- 检查设置页中 Screen Capture 权限状态

### 脚本执行报错

- 在日志页切换到"Error"过滤器查看错误详情
- 确认脚本中 `from autopilot import *` 已正确导入
- 检查坐标是否超出屏幕范围（使用 `screen_width()` / `screen_height()` 获取）
- 找图操作确认模板图片路径正确且文件存在

## 十一、插件开发指南（高级）

### 插件接口说明

插件需实现 `PluginInterface` 接口，包含以下方法：

| 方法 | 说明 |
|------|------|
| `onCreate(context)` | 插件加载时调用，用于初始化 |
| `onDestroy()` | 插件卸载时调用，释放资源 |
| `execute(params)` | 执行插件逻辑，接收参数 Map，可返回任意结果 |
| `getName()` | 返回插件唯一名称 |
| `getVersion()` | 返回插件版本号 |

### 创建插件 APK

1. 新建 Android Library 模块，依赖 MapleAuto 的 `plugin` 模块
2. 创建类实现 `PluginInterface` 接口
3. 在 `assets/plugin.json` 中声明入口类：

```json
{
  "main": "com.example.myplugin.MyPlugin",
  "name": "My Plugin",
  "version": "1.0"
}
```

4. 编译生成 APK 文件

### 加载与执行插件

将插件 APK 放置到手机存储中，MapleAuto 通过 `DexClassLoader` 动态加载插件 APK，读取 `assets/plugin.json` 获取入口类名，实例化后在沙箱环境中执行。插件运行在独立的类加载器中，与主应用隔离。
